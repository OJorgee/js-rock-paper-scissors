
# Rock - Paper - Scissors

A console based rock/paper/scissors game where you play against an evil AI, built with JavaScript. This project was developed as a technical assignment to demonstrate clean code principles, logic implementation, and effective error handling.

## Live Demo: [ojorgee.github.io/js-rock-paper-scissors/](https://ojorgee.github.io/js-rock-paper-scissors/)

## Features
* **Interactive Gameplay:** Challenges the player to play five rounds of Rock, Paper, Scissors against the AI.
* **Robust Error Handling:** Validates user inputs continuously, ensuring the game flow is never interrupted by invalid data.
* **Modular Logic:** Structured with distinct, reusable functions for random string generation, input validation, and game state management.

## How to Play
1. Open the live demo in your web browser: https://ojorgee.github.io/js-rock-paper-scissors/
2. Open the **Browser Console** (`F12` or `Right-Click` -> `Inspect` -> `Console`).
3. The game will prompt you to enter your option directly in the browser's input dialog.

## Technical Implementation
* **Language:** Vanilla JavaScript (ES6+).
